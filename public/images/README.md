# Website images — drop-in guide

Drop your final PNG files into these folders using the **exact filenames** below.
The website loads them automatically — no code changes needed. If a file is
missing, the site falls back to a tasteful branded placeholder, so nothing ever
looks broken while you're still adding photos. Replace a file and redeploy (push
to GitHub) and the new image appears everywhere it's used.

## hero/
- `florida-waterfront.png` — panoramic South Florida waterfront.
  Used on: homepage hero, section dividers, consultation CTA background, About
  page. A subtle dark overlay is applied automatically wherever text sits on it.
  Recommended: wide/panoramic, at least 2400px wide.

## practice-areas/  (one image per practice area)
- `business-litigation.png`
- `business-transactions.png`
- `community-associations.png`
- `hotel-motel-development.png`
- `information-technology.png`
- `title-insurance-closings.png`
  Used as large supporting imagery on each practice-area page (split layouts /
  banners) and in the homepage practice section. Recommended: landscape,
  ~1600×1200 or wider.

## attorneys/  (professional portraits, not avatars)
- `joshua-gerstin.png`
- `alicia-pokhoy.png`
  Used on the homepage attorney spotlight, Meet the Attorneys page, and each
  attorney's profile page. Recommended: portrait orientation, ~900×1100.
  (The current files are the official firm headshots — replace with your finals
  anytime.)

## office/
- Any filenames. The first image named `office-1.png`, `office-2.png`, … (or
  `reception.png`, `lobby.png`, etc.) is picked up on the Contact and About
  pages. Recommended: landscape, ~1600×1000.

## backgrounds/
- Any filenames (e.g. `texture.png`, `boca-skyline.png`). Used for decorative
  full-width background sections. Recommended: wide, high-resolution.

## about/
- Any filenames (e.g. `firm-1.png`, `team.png`). Used on the About page in
  editorial split layouts. Recommended: landscape.

---
Note: filenames are case-sensitive and must match exactly, including the `.png`
extension.
